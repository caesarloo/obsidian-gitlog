# 更新日志

本文件记录项目的所有重要变更。

## v1.0.2 - 2026-02-27

### 新增
- 新增 Windows 本地开发环境修复脚本（`npm run doctor:win`），可自动检查并修复常见命令可用性问题。

### 变更
- 优化 Git 状态路径解析逻辑，增强对重命名路径与带引号/转义路径的处理兼容性。
- 完善 Release 构建产物处理，构建后自动复制 `manifest.json` 与 `styles.css` 到 `dist` 目录。

### 文档
- 更新 Release 与本地环境修复相关说明，补充 Windows 场景下的故障排查指引。

## v1.0.1 - 2026-02-25

### 新增
- 优化更新摘要生成，支持“主题化标题 + 关键明细”的多文件总结方式。
- 新增基于文件路径关键词与扩展名的语义主题归纳能力。

### 变更
- 改进兜底摘要逻辑，使 PDF、XLSX 等非文本文件也能输出有意义描述。
- 优化摘要结构，增加变更类型统计与关键文件高亮。

### 修复
- 清理 `copilotService` 中未使用逻辑，满足更严格的 TypeScript 检查。
- 将浏览器专用的 `window` 日志访问替换为 `globalThis` 安全访问。
- 将 Vite 配置迁移到 ESM（`vite.config.mts`），消除 CJS Node API 弃用警告。

### 构建与验证
- 全量错误检查通过（无编译错误）。
- `npm run build` 构建通过。
