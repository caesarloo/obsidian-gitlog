# Obsidian GitLog 发布说明

## 版本信息
- 版本：`v1.0.1`
- 发布日期：2026-02-25
- 版本类型：功能优化 + 稳定性修复

## 主要更新

### 1) 更新摘要质量优化（重点）
针对多文件变更场景，优化了自动生成摘要过于泛化的问题。

- 过去：容易出现“批量更新 N 个文件”这类信息量较低的描述。
- 现在：改为“主题化标题 + 关键明细”输出，摘要更具体、更可读。
- 增加了基于文件路径/关键词/扩展名的语义归纳能力（如日志、评优材料、发票报销等）。
- 对 `PDF / XLSX` 等难以提取文本 diff 的文件，也能生成有意义的变更描述。

### 2) 摘要结构改进
- 标题由固定统计文案升级为按主题聚合（例如“工作日志更新、评优公示材料同步”等）。
- 详细描述加入变更类型统计（新增/修改/删除）与关键文件条目。
- 影响范围继续保留完整文件列表，便于追踪。

### 3) 代码质量与构建稳定性
- 清理 `copilotService` 中未使用的旧逻辑与冗余方法。
- 修复 `window` 类型引用导致的 TypeScript 问题，改为更通用的全局对象访问方式。
- 迁移 Vite 配置到 ESM 形式，消除构建中的 CJS Node API 弃用警告。

## 技术变更清单
